Detecção de Cola em Provas Escolares

 

I. Descrição do Problema

Cola de alunos em provas escolares é um assunto por demais corriqueiro, embora sumamente indesejável. Não há uma definição precisa de cola, mas supõe-se que haja cola toda vez que duas provas tenham um grau suspeito de semelhança. O tamanho da cola é variável: pode ser uma parte de uma questão, toda a questão, algumas questões, ou a prova inteira. Também, a cola pode ser grosseira (um simples copiar-colar), ou sutil (um copiar-colar parcial).

O que se quer é um sistema que detecte cola, de qualquer tamanho, ou tipo.

I.1 Uma Solução Não Aceitável

É claro que uma inspeção manual, por um professor ou monitor, etc., permite detectar cola. Mas é uma tarefa tediosa e de possivelmente alto custo, devendo portanto ser abandonada.

Para a solução do problema a um custo baixo, técnicas de mineração de dados devem ser investigadas, visando à indução automática de modelos de cola.

IV. Os Dados para Mineração

Você deve criar seus próprios dados. Sugestões: (1) conceba uma prova com 4 questões; (2) para facilitar, todas as questões são descritivas (ou somente texto); (3) suponha que aproximadamente 50 alunos fazem a prova; e (4) todos os tamanhos e tipos de cola são possíveis.
